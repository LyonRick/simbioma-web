# Sincronizar Documentación del Proyecto

Revisa y sincroniza la documentación del proyecto:

1. **Verificar Consistencia**
   - Lee el PRD: `docs/PRD-Simbioma.md`
   - Compara con implementación actual
   - Identifica discrepancias

2. **Actualizar Documentación Técnica**
   - Schema de base de datos en `docs/architecture/database-schema.md`
   - Endpoints de API en `docs/api/endpoints.md`
   - Componentes en README de cada directorio

3. **Verificar Brand Guidelines**
   - Confirma que colores en código coincidan con `design/brand/brand-guidelines.md`
   - Verifica uso correcto de tipografía
   - Revisa tono de textos en UI

4. **Reportar**
   - Lista de inconsistencias encontradas
   - Prioridad de actualización (Alta/Media/Baja)
   - Sugerencias de mejora

Guarda reporte en `.claude/outputs/reports/sync-docs-[fecha].md`
